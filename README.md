# Say-Hello-Alexa
* Type 'qianwei test' to start
* Say "hi"

  Device responds: "hello".
* Ask "Where is **#hallname** hall ? ". Where **#hallname** can be Gilman, Clark, Hodson, Mason, Levering, Krieger, Garland.

  Device responds the location.
* Say anything else

  Device responds " You said, '...' ".
